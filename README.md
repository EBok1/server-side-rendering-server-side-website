Ontwerp en ontwikkel een online platform waar buurtinitiatieven uit Oud-West een verzameld en bekeken kunnen worden. Het idee is dat het platform een soort marktplaats/vacature-achtige site moeten zijn waar vraag en aanbod bij elkaar komen, met ruimte voor uitbreiding van vraag en aanbod. Het is de bedoeling dat men contact met elkaar kan zoeken via de site.

## Gebruik
<!--Bij Gebruik staat hoe je project er uit ziet, hoe het werkt en wat je er mee kan. -->
Je kunt de site gebruiken en bezoeken via onderstaande link
Link: https://lovely-red-lab-coat.cyclic.app/ 

Wanneer je op de site komt, zie je als eerst de homepagina met een afbeelding. Wanneer je naar beneden scrolt zie je een stukje uitleg over wat de buurtinitiatieven inhouden. Als je verder naar beneden scrolt zie je een aantal voorbeelden van initiatieven. Daaronder staat nog een formulier waar je zelf een initiatief kan insturen. Verder is er op de pagina een navigatie en een footer te vinden. Via de menubalk kan je naar de buurtprojecten pagina, hier staan alle projecten. Als je op een item klikt, krijg je de detail pagina daarvan te zien. Je ziet een uitgebreide beschrijving over de vraag/aanbod. Deze pagina heeft nog geen styling en mist nog een aantal onderdelen.

## Installatie
<!-- Bij Instalatie staat hoe een andere developer aan jouw repo kan werken -->
1. Clone of download deze respository (fork)
2. Openen op je code editor

   2.1 Dit kun je doen bij < code > (github), open met github desktop
   
   2.2 Via Github Desktop kun je dit project dan kiezen
   
   2.3 Rechter muisknop > Open in Visual Stude Code, bijvoorbeeld (code editor)
   
3. Voordat je iets kunt veranderen, moet je eerst `Node js` instaleren, omdat we met een API werken
 3.1 Installeer: nodejs.org (Node 20.11.0 LTS,)
4. Wanneer je dit geinstaleerd hebt moet je hem nog in je code editor installeren
5. Open terminal, en type `npm install` > installeerd `Node js`
6. Om de site te bekijken, type je in de terminal: `npm start`
7. Wanneer je wijzigingen aanbrengt in de `server.js`, moet je in de terminal het resetten met `control c` en daarna weer `npm start` om het project te bekijken
